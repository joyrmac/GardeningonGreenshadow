## SIDE VIEW: DOME SHAPE (C-SHAPED OPENING SOUTH)

```
                      NORTH
                       ↑

           TALL PEAK (center-north)
                     / \
                  /       \
               /             \
            /    MEDIUM        \
         /        ZONE           \
       /                           \
      /  OUTER RING                 \
     /  (short edges)                \
    ├─────────────────────────────────┤

    This creates a protective dome
    opening south to full sunlight

    North side: Taller plants shelter
    South side: Open to maximum light

    Effect: Center + outer ring get
    direct southern sun year-round
```

## SPACING REFERENCE: COMMON MISTAKES TO AVOID

### ❌ DON'T DO THIS (Too crowded)

```
• • • • • • • • •
(Plants touching = disease, poor air flow)
```

### ✓ DO THIS (Proper spacing)

```
•          •          •          •
(Allow growth room, air circulation)
```

### SPACING CHART

| Plant | Spacing | Reason |
|-------|---------|--------|
| Valerian | 24" | Tall + spreads laterally as matures |
| Echinacea | 12-18" | Medium spread, needs air flow |
| Yarrow | 12" | Spreader, needs thinning room |
| Hyssop | 12" | Bushy, spreader |
| Chamomile | 10-12" | Heavy self-seeder, thin continuously |
| Calendula | 10" | Bushy annual |
| Fenugreek | 6-8" | Tight spacing = bushier plant |
| Borage | 12" | Spreader, needs room |

## THE "C" EXPLAINED: WHY THIS SHAPE?

```
CLOSED CIRCLE (old idea)
                ↓
        Shaded center
        Some sun blocked
        by north tall plants
        Stilted view

              vs.

C-SHAPED OPENING SOUTH (new idea)
                ↓
        Maximum southern
        sun reaches center
        EVERY season

        View from
        standing spot = dramatic

        Tall northern plants =
        wind shelter

        Dome shape obvious
        from above/side

        Sacred geometry visible:
        concentric circles radiating
        from standing point
```
