# 📦 Archive — Historical Planning & Experiments

This folder contains **old drafts, superseded plans, and experimental ideas** from the Greenshadow Garden planning process.

**You don't need these to run the garden.** Everything essential has been consolidated into the active guides.

---

## Why This Archive Exists

Gardening is iterative. Plans evolve. Some ideas work, some don't. This archive preserves:
- **Planning history** — why current beds exist in their current form
- **Experimental iterations** — different layout attempts before landing on final plans
- **Abandoned ideas** — what was considered but never built
- **Old inventory systems** — how the plant list evolved
- **One-off planning docs** — specific dates and task lists that have passed

## Archive Cleanup Strategy

Over time, we'll:
1. **Consolidate truly duplicate files** into single versions
2. **Delete social media hashtags and one-off docs** when they become obsolete
3. **Keep design/research docs** as reference
4. **Move truly useful guides** (like Garden_Photography_Guide) to active reference if needed
5. **Date files** so we know how old they are

Current state: **Everything preserved as-is** for historical reference.
