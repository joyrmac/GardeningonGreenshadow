## Quick Facts

- **Size:** 8x8' square bed
- **Center feature:** Tulip magnolia (deciduous, creates summer shade)
- **Sun:** Partial shade (summer) to more light (winter)
- **Key plants:** Catnip, cat mint, cat grass, + NC native perennials around edges
- **Layout:** Sand potty area in middle, native perennials on perimeter for pollinator value
- **Philosophy:** Contained growth, ecosystem benefit, shade-tolerant

---

## Perimeter Plants (NC Natives, Shade-Tolerant)

- **North:** Aromatic Aster (fall bloomer)
- **East:** Virginia Mountain Mint (contained, pollinator magnet)
- **South:** Blue Mistflower (soft blue, shade-tolerant)
