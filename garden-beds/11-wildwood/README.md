## How to Use These Documents

### **For Planning & Design**

1. Read **Joy's Chart** to understand what you need emotionally/creatively
2. Read **Garden Philosophy** to see how those needs translate to design
3. Review **Master Plan** to see the full Wildwood layout
4. Consult **Color Palette Guide** to finalize aesthetic choices

### **For Building the New Bed**

1. Read **Bed Specifications** cover-to-cover (it's complete)
2. Check **Building Timeline** (March for prep, May for planting)
3. Use **Materials List** for shopping
4. Follow **Step-by-Step Instructions** for build
5. Reference **Planting Calendar** for timing
6. Use **Maintenance Schedule** through the season
7. Record observations in **Journal Template**

### **For Daily Garden Work**

1. Consult **Master Plan** for which bed you're working on and its purpose/colors
2. Check **Bed Specifications** for watering/fertilizer/harvest timing
3. Reference **Color Palette Guide** for companion planting harmony
4. Check **Maintenance Calendar** for what's due this week
