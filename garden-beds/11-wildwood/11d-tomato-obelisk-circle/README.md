## Getting Started

1. Read **Potion_Patch.md** for overview
2. Choose layout: concentric rings or compass/wedge design
3. Follow **Potion_Patch_Planting_Guide.md**
4. Use **Potion_Patch_Position_Guide.md** for placement
5. Track progress with **Potion_Patch_Position_Checklist.md**

## Herbs Included

Medicinal herbs only (no culinary varieties):
- Lavender, Feverfew, Valerian, Passionflower, Skullcap
- Holy Basil, Ashwagandha, Gotu Kola, Ginseng, Elecampane
- Echinacea, Nettle, Red Clover, Dandelion, Burdock
- Bee Balm, Bergamot, Catmint, Self-heal
- Chamomile, Calendula, St. John's Wort, Yarrow, Wood Betony
- Hyssop, Germander, Santolina, Agrimony, Betony
- Mullein, Comfrey, Plantain, Horehound, Motherwort
- Rue, Wormwood, Artemisia, Mugwort, Southernwood
