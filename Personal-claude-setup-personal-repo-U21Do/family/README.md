## 💡 Core Principles

✅ **Consistency over perfection.** Same bedtime, same chores, same Sunday routine.

✅ **Each kid is different.** Different expectations, different approaches, different triggers.

✅ **Structure + flexibility.** Rules matter. Grace matters too.

✅ **Simplify when overwhelmed.** Drop what's not essential. Add it back later.

✅ **Sunday rest is sacred.** Non-negotiable. It's why the week works.

✅ **Kids need to be seen.** Weekly sync, celebration of wins, understanding of who they are.

✅ **You're not failing.** You're showing up, trying, and that's enough.
