# Joy McNamara - Personal Information Repository

Your personal knowledge base — organized, searchable, and cross-referenced. Every folder now has a **markdown overview** that makes its contents searchable and summarized, while preserving the original PDFs and data files.

---

## How This Repo Works

Each folder contains:
1. **Original files** (PDFs, CSVs, JSON, TXT) — your source documents
2. **A markdown overview** (`*_Overview.md` or similar) — searchable summary of what's in the folder, key facts extracted, and cross-references to related content elsewhere
3. **How-to guides** — practical, ADHD-friendly guides that teach you how to use the information in each section

Start with the markdown file in any folder to get oriented. Dig into the originals when you need detail. Read the guides when you want to learn.
